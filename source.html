Also known as "serverless", "client-side", or "static" web apps, unhosted web apps do not send your user data to their server. You connect your own server, or your data stays within the browser.

advantages compared to native apps

  • they work on any device, any manufacturer, any platform, any provider
  • anybody can run an unhosted web app store (unless you use Chrome)

unhosted web apps compared to hosted web apps

  • choose your server provider independently of your which apps you use
  • likewise, choice of apps is not narrowed down by your server provider
  • the app provider no longer gets to see your data by default
  • unhosted web apps are cheaper to publish than hosted ones
  • unhosted web apps can easily be mirrored for kill-switch resilience
  • infinite per-app backend scalability (there simply is no per-app backend)
